# jaemincho.github.io

Personal portfolio site for Jaemin Cho — Mechatronics & AI Engineering student.

Live at: [jaemincho.github.io](https://jaemincho.github.io)

## Structure

```
├── index.html        # Page content
├── css/style.css     # All styles (CSS variables at top for easy theming)
├── js/main.js        # Mobile nav + scroll behavior
└── assets/           # Images
```

## Making updates

- **Add a project** — copy a `.project-card` block in `index.html` under the projects section
- **Add a job** — copy a `.timeline-item` block in the experience section
- **Add a skill** — add a `<span class="skill-tag">` in the relevant skill category
- **Change colors** — edit the CSS variables at the top of `css/style.css`
